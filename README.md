#  Panela El Manzano

Proyecto web interactivo creado con **HTML, CSS y JavaScript**, que promueve productos artesanales de *Panela El Manzano*.  
Incluye animaciones, interactividad, efectos visuales y enlaces directos a WhatsApp y redes sociales.



##  Características
- Barra de navegación animada
- Video de fondo con overlay y CTA de compra
- Línea de tiempo del proceso de fabricación
- Animaciones con **LottieFiles**, **ScrollMagic** y **Animate.css**
- Pie de página con enlaces sociales y mapa embebido de Google Maps



##  Tecnologías usadas
| Tecnología | Descripción |
|-------------|-------------|
| **HTML5** | Estructura semántica y SEO básico |
| **CSS3 / Bootstrap 5** | Estilos, colores y diseño responsivo |
| **JavaScript** | Interactividad, animaciones, audio y lógica de compra |
| **LottieFiles & Animate.css** | Animaciones modernas y suaves |
| **Google Maps API** | Mapa embebido con ubicación del negocio |
